## Set up the codebase for dev

### 1 set up poetry env

first install poetry

```bash
pip install -U poetry
```

You don't need to setup a virtual env, poetry will take care of it.


```bash
poetry install
```

then activate the virtual env

```bash
poetry shell
```


### 2 pre commit

install pre commit hook

```bash
pre-commit install
```
